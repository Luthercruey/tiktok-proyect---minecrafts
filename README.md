# Minecraft TikTok Panel

## Descripción
Esta aplicación desarrollada con Electron permite mostrar un panel interactivo para streams de Minecraft en TikTok, mostrando eventos y mejorando la interacción en vivo.

Ademas de eso facilitando su uso para gente con pocas habilidades en programación haciendolo mucho más accesible para todo publico como tambien una herramienta de entretenimiento.

## Cómo ejecutar el proyecto

1. Clonar el repositorio:
git clone https://github.com/Luthercruey/tiktok-proyect---minecrafts.git

2. Entrar a la carpeta:
cd tiktok-proyect---minecrafts

3. Instalar dependencias:
npm install

4. Ejecutar la aplicación:
npm start

## Estructura del proyecto

- src/main → Backend
- src/renderer → Frontend
- src/assets → Recursos
